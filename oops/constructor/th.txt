constructor:-
constructor is a special function which has
same name as class name...
constructor soes not have return type...
but constructor can have params...

type of constructor:-

1. default constructor
2. parameterized constructor
3. copy constructor

use:
1. to initialize the data members of the class // instance variable
2. to allocate memory to the object
3. to perform some initialization tasks

one class required min one constructor...

if you dont create any constructor 
then compiler will create default constructor for you...
